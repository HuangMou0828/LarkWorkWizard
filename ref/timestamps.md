# 时间戳计算

飞书项目的日期字段使用**毫秒时间戳**，对应 CST（UTC+8）当天 00:00:00。

## 计算公式（优先使用）

```
目标日期 00:00 CST 的毫秒时间戳:
  ms = Date.parse('YYYY-MM-DDT00:00:00+08:00')

结束时间 23:59:59:
  ms_end = ms + 86_399_000
```

手动推算：

```
基准：2026-01-01 00:00:00 CST = 1,767,225,600,000 ms
N 天后 = 基准 + N × 86,400,000
```

## 快速诊断

| 年份 | 时间戳量级 |
|------|-----------|
| 2025 | `1,745,xxx,xxx,000` |
| 2026 | `1,767,xxx,xxx,000` ~ `1,798,xxx,xxx,000` |
| 2027 | `1,798,xxx,xxx,000` ~ `1,830,xxx,xxx,000` |

如果计算结果的量级与当前年份不符，说明基准年份用错了。
